# Opera Mini的一些特殊页面

server:reset  
重置 cookies

server:version  
效果等于输入 debug: ，但有更详细的服务器信息。

debug:  
除错页面

debug:console  
ECMAscript 控制台。

myopera:login  
登录到 Opera Link 页面，登录后就可以同步 OM 的书签到服务器。

myopera:signup  
注册 Opera Link

operette:/message?t=TITLE&m=MESSAGE  
提交反馈。把 TITTLE 和 MESSAGE 发送给 Opera。

opera:about  
关于页面。相当于 about: 。

opera:config  
服务器配置。相当于 config: 。

下面是一些本地页面，注意大小写：

o:A  
Home ，就是打开 OM 之后看到的界面。

o:B  
Main Menu ，主菜单。

o:C  
"Tools" menu，工具菜单。

o:I  
（是大写的 i）Settings，设置页面。相当于按下 #8 。

o:J  
Help，打开帮助。

o:L  
Help -> Hotkeys，打开帮助里的快捷键项目。

o:M  
Page Information，查看页面信息，相当于按下 #6。

o:V  
History，打开历史记录，相当于按下 #5。

o:X  
Help -> About，打开关于页面。

o:Y  
Add bookmark，添加书签界面，相当于按下 #7。

o:Z  
Bookmarks，书签界面，相当于按下 #2。

o:d  
和 o:Y 一样，添加书签。

o:h  
Help -> About -> Reset the counter is loaded，把流量统计的计数清零。

o:m  
Search，搜索界面，相当于按下 #9。
